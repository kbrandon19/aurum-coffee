# ☕ Aurum Coffee

A modern, headless, content-driven basic coffee brand website built with **Next.js**, **Sanity**, and **Plasmic**.

This project demonstrates how to combine a CMS (Sanity), a visual UI builder (Plasmic), and a React framework (Next.js) to create a dynamic, scalable web experience.

[View Website](https://aurum-coffee.plasmic.run/)



---

##  Tech Stack

- **Next.js (App Router)** – frontend framework
- **Sanity** – headless CMS for coffee products
- **Plasmic** – visual UI builder for pages and components
